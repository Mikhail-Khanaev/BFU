Методы сортировки. Сортировка методом причёсывания

Дана последовательность чисел. Отсортировать и вывести последовательность чисел, определённым методом

Принцип алгоритма

Сортировка методом причёсывания представляет собой улучшенный вариант сортировки пузырьком

Сортировка пузырьком основана на сравнении и перестановке соседних элементов, то есть разница их индексов всегда равна 1. Но если увеличить ее и сравнивать элементы стоящие на большем расстоянии друг от друга, то минимальные элементы, находящиеся в конце массива, можно передвинуть в его начало за меньшее количество перестановок

На каком же расстоянии стоит сравнивать элементы?

Во первых, это число зависит от длины самого массива — чем больше массив, тем больший разрыв можно брать между элементами

Во вторых, это расстояние должно постепенно уменьшиться до 1, чтобы на последней итерации цикла мы смогли проверить все элементы один за другим как в обычной сортировке пузырьком

Это хорошо видно на иллюстрации:

Число, на которое должен раз за разом уменьшаться разрыв, было названо фактором уменьшения и выведено авторами сортировки из формулы. Оно равно ~1.247. В своей программе я буду использовать именно его, но можно просто каждый раз уменьшать шаг на 1. Поэтому за первоначальный шаг я беру не длину массива минус один, а длину массива, деленного на фактор уменьшения и округленного до целого. После каждой итерации я так же буду делить на этот фактор уменьшения и округлять. Это будет продолжаться до тех пор, пока шаг не станет равным 1

                                                                function CombSorting(array) {
    let l = array.length;
    let factor = 1.247;
    let gapFactor = l / factor;

    while (gapFactor > 1) {
        let gap = Math.round(gapFactor); // тут нужно округлять до целого,
                                         // так как мы делим наш шаг на 1.247, 
                                         // что в итоге даёт нецелое число, 
                                         // а шаг таким быть не может

        for (let i = 0, j = gap; j < l; i++, j++) {
            if (array[i] > array[j]) {
                [array[i], array[j]] = [array[j], array[i]];
            }
        }
        gapFactor = gapFactor / factor;
    }
    return array;
};

                                        # полученную строку преобразуем в массив чисел
nums = list(map(int, input("Введите числовую последовательность\n").split(" ")))

step = len(nums) - 1

while step > -1:
    for i in range(len(nums) - step):
        if nums[i] > nums[i + step]:
            nums[i], nums[i + step] = nums[i + step], nums[i]
    step -= 1

print("Отсортированная последовательность")
print(*nums)
