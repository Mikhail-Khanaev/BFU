Задача о скобках

На вход подаётся строка, состоящая из скобок. Программа должна определить правильность введёного скобочного выражения. Программа должна работать на русском языке: “Введите строку”, “Строка не существует”, “Строка существует” и т.п.

Пункт 1: В строке будут скобки только одного типа: или “()”, или “{}”, или “[]”

Пункт 2: В строке будут все три вида скобок

Принцип алгоритма

Сначала мы перебираем каждый элемент строки. Если нам попадается открывающаяся скобка, то мы добавляем её в стек. Стек это такая структура даннных, которая работает по приципу "последним зашёл, первым ушёл". Если нам попадается закрывающаяся скобка, то из стека мы достаём открывающаюся. Если там эта скобка есть, то всё отлично - мы нашли пару скобок. Но если этой скобки там нет, то значит, что стек уже пустой. Следовательно в строке есть лишние скобки. Функция завершается. Но если интерация закончилась и функция не завершилась, то мы проверяем, не пустой ли у нас стек. Если в стеке что-то есть, то значит, что у нас открывающихся скобок было больше и строка неправильная. Функция также завершается. Но если и это условие не выполнилось, то значит, что всё отлично.

                                        function checkBrackets(str) {
    let stack = [];
    let brackets = ["[", "(", "{"];
    let obj = {
        "{" : "}",
        "(" : ")",
        "[" : "]",
    }

    for (let i = 0; i < str.length; i++) {
        const bracket = str[i];

        if (brackets.includes(bracket)) {
            stack.push(bracket);
        } else {
            const lastElement = stack.pop();

            if (bracket !== obj[lastElement]) return false;
        }
    }

    if (stack.length !== 0) return false;
    return true;
}

                                        while True:
    line = input("Введите строку:\n")
    opposites = {
        ")": "(",
        "}": "{",
        "]": "["
    }

    result, storage, count = True, [], -1

    for char in line:
        if char in opposites.values():
            storage.append(char)
            count += 1
        else:
            if count == -1:
                result = False
                break
            count -= 1
            if storage.pop() != opposites[char]:
                result = False
                break

    if result and count == -1:
        print("Строка является правильной скобочной последовательностью")
    else:
        print("Строка не является правильной скобочной последовательностью")
