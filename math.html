Таблица производных

\((x^n)^{\prime} = nx^{n-1}\)

\((a^x)^{\prime} = a^x\cdot ln(a)\)

\((e^x)^{\prime} = e^x\)

\((\log_a x)^{\prime} = \frac{1}{x\cdot ln(a)}\)

\((\ln x)^{\prime} = \frac{1}{x}\)

\((sin(x))^{\prime} = cos(x)\)

\((cos(x))^{\prime} = -sin(x)\)

\((\sqrt{x})^{\prime} = \frac{1}{2\sqrt{x}}\)

\((tg(x))^{\prime} = \frac{1}{cos^2(x)}\)

\((ctg(x))^{\prime} = -\frac{1}{sin^2(x)}\)

\((arcsin(x))^{\prime} = \frac{1}{\sqrt{1-x^2}}\)

\((arccos(x))^{\prime} = -\frac{1}{\sqrt{1-x^2}}\)

\((arctg(x))^{\prime} = \frac{1}{1+x^2}\)

\((arcctg(x))^{\prime} = -\frac{1}{1+x^2}\)

\((sh(x))^{\prime} = ch(x)\)

\((ch(x))^{\prime} = sh(x)\)

\((th(x))^{\prime} = \frac{1}{ch^2(x)}\)

\((cth(x))^{\prime} = -\frac{1}{sh^2(x)}\)

Таблица интегралов

\(\int x^n\;\mathrm{d}x = \frac{x^{n+1}}{n+1} + C\)

\(\int \frac{\mathrm{d}x}{x}\;=ln|x| + C\)

\(\int a^x\;\mathrm{d}x = \frac{a^x}{ln(a)} + C\)

\(\int e^x\;\mathrm{d}x = e^x + C\)

\(\int cos(x)\;\mathrm{d}x = sin(x) + C\)

\(\int sin(x)\;\mathrm{d}x = -cos(x) + C\)

\(\int \frac{\mathrm{d}x}{cos^2(x)}\; = tg(x) + C\)

\(\int \frac{\mathrm{d}x}{sin^2(x)}\; = -ctg(x) + C\)

\(\int \frac{\mathrm{d}x}{\sqrt{1 - x^2}}\; = arcsin(x) + C\)

\(\int \frac{\mathrm{d}x}{1 + x^2}\; = arctg(x) + C\)

\(\int sh(x)\mathrm{d}x = ch(x) + C\)

\(\int ch(x)\mathrm{d}x = sh(x) + C\)

\(\int \frac{\mathrm{d}x}{ch^2(x)}\; = th(x) + C\)

\(\int \frac{\mathrm{d}x}{sh^2(x)}\; = -cth(x) + C\)

Эквивалентности

\(sin(x) \sim x\)

\(tg(x) \sim x\)

\(1 - cos(x) \sim \frac{1}{2} x^2\)

\(e^x - 1\sim x \)

\((1+x)^n - 1\sim nx\)

\(\sqrt{1+x} - 1 \sim \frac{1}{2}x\)

\(arcsin(x) \sim x\)

\(arctg(x) \sim x\)

\(ln(1 + x) \sim x\)

\(a^x - 1\sim xln(a) \)

\(\sqrt[n]{(1+x)} - 1\sim \frac{1}{n}x\)

\(\sqrt[3]{(1+x)} - 1\sim \frac{1}{3}x\)

Ряды Тейлора

\(e^x = 1 + x + \frac{x^2}{2!} + \dots + \frac{x^n}{n!} + \dots \)

\(ln(1+x) = x - \frac{x^2}{2} + \frac{x^3}{3} - \dots + (-1)^{n-1}\cdot \frac{x^n}{n} \)

\(sin(x) = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \dots + (-1)^{n-1}\cdot \frac{x^{2n-1}}{(2n - 1)!} \)

\(cos(x) = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \dots + (-1)^{n-1}\cdot\frac{x^{2n-2}}{(2n - 2)!} \)

\(arctg(x) = x - \frac{x^3}{3} + \frac{x^5}{5} - \dots + (-1)^{n-1}\cdot\frac{x^{2n-1}}{(2n-1)} \)

\(arcsin(x) = x + \displaystyle\sum_{n=1}^{\infty}\frac{(2n - 1)!! \cdot x^{2n+1}}{(2n)!!\cdot (2n+1)}\)

\(sh(x) = x + \frac{x^3}{3!} + \frac{x^5}{5!} + \dots + \frac{x^{2n-1}}{(2n-1)!} + \dots \)

\(ch(x) = 1 + \frac{x^2}{2!} + \frac{x^4}{4!} + \dots + \frac{x^{2n}}{(2n)!} + \dots \)

\(\frac{1}{1+x} = 1 - x + x^2 - x^3 + \dots + (-1)^n\cdot x^n + \dots\)

\(\frac{1}{1-x} = 1 + x + x^2 + \dots + x^n + \dots\)

\begin{equation} (1+x)^m = 1 + \frac{m}{1!}x + \frac{m(m-1)}{2!}x^2 + \dots + \end{equation}

\begin{equation} +\frac{m(m-1) \dots (m - (n-1))}{n!}x^n + \dots \end{equation}
